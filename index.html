About Doug Brown

Doug Brown, IT consultant, computer service and web developer. Have had the opportunity to build a number of websites using HTML, CSS, ASP (Active Server Pages), Visual Basic, some .Net, some PHP and Coldfusion. With all of the websites, they have always been data driven sites and pages, from the wording on the site pages to the data that you are searching for. I currently work for US Auto Finance, where I photo all of the inventory and post it to our website and other web advertising companies. Prior to US Auto Finance, I worked for Masterpiece Classic Cars and Masterpiece Vintage Cars. With Masterpiece Classic Cars I was the initial designer and web developer who built the website and designed the SQL database which housed all of the links and information pulled to display all of our inventory.

I am currently going to the Butler Executive Coding Boot Camp 6 month certification program working towards becoming a more proficient Full Stack programmer.
